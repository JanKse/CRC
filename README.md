### Validácia vstupov

Na encode a decode vstupy môže uživateľ zadať len 0 a 1. Iné mu program ani nedovolí zadať.

Na vstup Code môžete zadať len čísla od 0-1 a čiarku ",". Iné mu program ani nedovolí zadať.
